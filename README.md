## 📌 Project Overview

**Qurbani Hat** is a modern livestock marketplace designed to simplify the process of buying animals for Qurbani. Users can browse available livestock such as cows and goats, view detailed information, and place bookings after authentication. The platform focuses on a clean UI, smooth user experience, and secure user interaction.

---

## ✨ Key Features

* 🔍 Browse livestock (cows, goats, etc.)
* 📄 View detailed animal information
* 🔐 User authentication (login/register)
* 🛒 Booking functionality for selected animals
* 📱 Responsive design for mobile, tablet, and desktop
* ⚡ Fast and optimized performance

## 🛠️ Technologies Used

* **Frontend:** React.js
* **Routing:** React Router
* **Styling:** Tailwind CSS / CSS3
* **Authentication:** Firebase Authentication
* **Hosting:** Firebase Hosting
* **State Management:** Context API

---

## 🔐 Authentication Flow

* Users must log in or register before booking an animal
* Protected routes restrict unauthorized access
* Firebase handles secure authentication

## 🚀 How to Run Locally

```bash
# Clone the repository
git clone https://github.com/Nahid711811/qurbani-hat-project.git

# Navigate into the project
cd qurbani-hat-project

# Install dependencies
npm install

# Start the development server
npm run dev
```
